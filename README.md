# Potato Disease Classification using Deep Learning

This project uses Deep Learning and Convolutional Neural Networks (CNN) to classify potato leaf diseases.

## Classes
- Early Blight
- Late Blight
- Healthy

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Matplotlib

## Project Description
The model is trained on potato leaf images to detect diseases automatically. 
It helps farmers identify plant diseases early and take preventive actions.

## Dataset
PlantVillage Dataset (Potato Leaves)

## How to Run
1. Open the notebook in Google Colab
2. Install required libraries
3. Run all cells to train the CNN model
